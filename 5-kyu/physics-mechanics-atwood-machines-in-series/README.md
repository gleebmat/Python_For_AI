One of these masses can be replaced by another Atwood machine, and again, and again...</p><p>Write a function that receives a list of 2 to 100 masses (kg) and returns the acceleration (m/s/s) of the first mass.</p><h1 id="quick-physics-reference">Quick Physics Reference</h1><p>If you would like to test your Mechanics skills, this is all you need:</p><ul><li>Force = Mass x Acceleration</li><li>Gravitational Force (N) = Mass (kg) x 9.80665 (m/s/s)</li><li>Tension Force is equal for objects on two ends of the same string</li><li>Find acceleration by combining all forces on an object</li><li>In a many-mass system, multiple masses can often be reduced to one "effective" mass</li><li>Solve the two-mass system first, then replace the second mass with a pulley (recurse)</li></ul><p>Good luck!</p><h1 id="complete-physics-walkthrough">Complete Physics Walkthrough</h1><p>If you would like substantial help, read on! A many-mass solution can be derived recursively from the two-mass solution.
